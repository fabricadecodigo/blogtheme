#Introduction

Divine is an extremely beautiful theme with the unlimited color scheme, the elegant design, the friendly layout, and especially it has so many useful features.

Additionally,Divine attracts users by a lovely appearance, and one more interesting thing is that it is combined with Parallax effect, which helps sliders move at different speeds to create a sensation of depth. Not only is Divine Theme integrated with many plugins but also it is also compatible with Woo-Commerce Plugin- one of the most popular plugin to create online shop website.

Our detail documentation, clear tutorial video and demo.xml data file will help you install easily.

The theme was designed and coded by KOPA THEME

##Free freatured

- Fully responsive layout & Retina ready: Divine theme can work well on many different kinds of screen, resolution ranging PC, laptops from smart phones. Surely, you can experience with the most perfect way although what device is used.

- HTML5- CSS3: The theme is built using the latest web development technologies, including HTML5, CSS3 and jQuery.

- Translation ready: Language Localization supports .po/.mo file , so you can translate language into your language.

- Unlimited color scheme: Custom color/background on every front page section and on the entire page.

- Custom font: You can select font face, size, weight, line height and transform for every site element.

- Unlimited sidebar: Custom unlimited sidebar can be created and used.

##Premium Featured

- This theme support 630+ Google webfont

- Parallax Effect:Includes parallax effects that are able to customize color, add background in order that sliders are nicely displayed.

- Flexible Portfolio: Unlimited Portfolio section/ page allows you to create multiple Portfolio page with many different tags and full of functionality.

- Revolution Slider Plugin: Once you purchase this theme, you have license for Revolution Slider cost $18.

- Event Plugin:Display event list and event list with time-line layout and load-more button.

- Staffs Plugin: Customers easily see information of the companyâ€™s staffs, their position in the company, therefore , customers can give their opinions, questions to the personal with proper specialization so that he can reply correctly. - Testimonials Plugin:This is a good plugin to display the feedback of customers.

- Woo-commerce integration: Woo Commerce plugin allows you to create an online shop on your website.

- Useful Shortcodes:The Theme includes many useful shortcodes such as toggle, accordion, button, pricing table, drop cap with various styles for you to choose.

- Custom widgets: With many multi-function and useful widgets such as Kopa Dynamic Sidebar, Kopa Section, Kopa services, Testimonials, event time-line, icon pickers for the title and the effect for widgets, Divine theme will help you to build a really beautiful and professional website.
